# SimpleAI

A tiny character-level text generator built from scratch to learn how AI models work.

## What is this?

This project builds a small neural network (248,880 parameters) that:
1. Reads a text file of 99 inspirational quotes (6,201 characters)
2. Learns the patterns in it (which characters tend to follow which)
3. Generates new text that mimics the style

It's the same core idea behind ChatGPT and Claude — just massively scaled down to run on any laptop.

## How to run

```bash
# 1. Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Train the model (~30 seconds)
PYTHONPATH=src python3 src/train.py

# 4. Generate text interactively
PYTHONPATH=src python3 src/interactive.py
```

## Key concepts covered

- **Tensors** — the fundamental data structure of AI
- **Tokenization** — converting text to numbers and back
- **Neural networks** — embedding, RNN, and linear layers
- **Training** — forward pass, loss, backpropagation, optimizer
- **Epochs and batches** — how training data is organized
- **Loss curves** — visualizing whether training is working
- **Inference** — using a trained model to generate text
- **Temperature** — controlling randomness in generation
- **Hyperparameter tuning** — how data size, model size, and epochs affect results

## Architecture

```
"The only way..." → [19,29,26,1,...] → Embedding(48,128) → RNN(128,256,2) → Linear(256,48) → "e" (next char)
     text              numbers           128-dim vectors     256-dim context     48 scores        prediction
```
